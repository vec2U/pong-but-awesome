Stephen Lucas, Lead Creative Progammer for Composition Studies at UNT created this game and programmed the code in HTML, CSS, and JavaScript. I made this game as early-2000s Windows-Movie-Maker-YouTube-video-style as I could. I hope that the best adjective you can think of to describe this game is "baller." W and S control the left paddle; the up and down arrows control the right paddle. No, I don't know why "You're Gonna Go Far, Kid" cuts off halfway through.
